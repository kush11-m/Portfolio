## üöÄ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/kush11-m/Portfolio.git
cd Portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## üì¶ Build for Production

```bash
npm run build
```

The optimized production build will be in the `dist` folder.

## üé® Design System

### Colors
- **Background:** `#0a0a0a` / `#1a1a1a`
- **Text:** `#ffffff` / `#cccccc`
- **Accent:** `#007744` (dark green)
- **Borders:** `#ffffff` / `#444444`

### Typography
- **Headings:** Jersey 15 (pixel font)
- **Body:** Poppins (400-800 weights)

### Shadows
All elements use blocky pixel shadows with no blur:
```css
box-shadow: Npx Npx 0 #007744;
```

## üìß Contact Integration

The contact form uses FormSubmit.co for serverless form handling. Messages are sent directly to the configured email address with a custom thank-you modal.
